### 【タイトル・メタデータ】

- [ ]  タイトルは40-50文字（重要なキーワードを左側に配置）
- [ ]  ディスクリプションは100-120文字（検索意図に応える内容）
- [ ]  記事内容がひと目でわかる簡潔な表現

### 【本文・ライティング】

- [ ]  **ファーストビュー**（スクロール前）に解決策やキーワードが含まれている
- [ ]  見出し（H2/H3）を適切に使い、構造化されている
- [ ]  重要な箇所を太字（`<strong>`）や箇条書きで装飾している
- [ ]  スマホで見た時に「一画面が文字で埋め尽くされていない」か（改行の調整）
- [ ]  誤字脱字はないか

### 【画像関連】

- [ ]  **alt属性**（代替テキスト）に画像の内容を記述している
- [ ]  ファイル名は `seo-checklist.webp` のように英単語で意味を持たせる
- [ ]  拡張子は **WebP**
- [ ]  容量は **100KB以下**
- [ ]  横幅は最大 **1200px** 程度（レスポンシブ対応を確認）
