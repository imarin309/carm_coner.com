楽園追放というアニメ映画が原作のキットで、アンジェラさんはヒロイン兼主人公になります。
僕は元々このアニメをあまり知らず、パッケージの見た目がかわいいという理由で買いました。
:sm[金髪でクールな表情をしている女の子には弱い]

素組みしているときに原作が気になってアニメを見たのですが、、:xl[:red[これがめちゃ面白い！！！！]]

肉体を捨てて電脳化して暮らす近未来の世界で生きるアンジェラが、現在の我々が暮らす地球で生活をする男ディンゴと出会い価値観を揺るがしていく。
:large[僕はこういう電脳な世界観のアニメ大好き。]

そしてヒロインのアンジェラさんは原作を見てさらに好きになりました。
クールに仕事をこなしていて一見頭が硬そうに見えるも、新しく出会った世界にも柔軟に適応しようと努力している姿が魅力的。
そしてツンデレ、声優さんは釘宮理恵さん。

:large[:red[これは平成生まれのオタクはみんな好きですよ！！]]

ぜひ原作もみんな見てほしい。

話それましたが、そんなアンジェラさんの美プラを制作していきたいと思います。

今回は色を変えたりしないで純粋に原作通り塗っていきたい。
アンジェラさんのこのキットで一番目立つ長く美しい髪をハイライトでぼかしたりしてアニメ調っぽくする！

## 塗装

水性塗料ファレホで筆塗り塗装します。
先ほども書きましたが、特にチャレンジしたい点は髪にハイライトをつけるところ。

手順

1. 下地を塗る
2. 髪を塗る
3. 肌を塗る
4. 服を塗る
5. トップコートを塗る

今まで塗装をするときに同じ色の部分を塗って、その次に違う色を、、という感じに塗ってきたのですが、今回は部位ごとに塗ってみることにしました。

### 髪を塗る

最初に髪を塗ります。

  images={[
    { src: "https://assets.calm-corner.com/posts/angela-balzac/anjela_balzac_paint_create_color.webp", alt: "アンジェラバルザック_塗装_髪の色を作る", caption: "髪の色を作る" },

まず素のアンジェラちゃんぽい色を作り、光が当たっていそうなところに少し明るい色を塗る。で、その明るい色のところにさらに明るく調色した色を塗る、、といったことを4回繰り返す、、。

### 肌を塗る

次に肌を塗ります。

<ImageGallery
  images={[
    { src: "https://assets.calm-corner.com/posts/angela-balzac/anjela_balzac_paint_skin.webp", alt: "アンジェラバルザック_塗装_肌の色を作る", caption: "肌の色を作る" },
  ]}
/>

ここは結構気を使いました！！

今まで肌色一辺で塗装をしたときに、肌の色として血の通っていないような違和感を感じていたんですよね。
で、今回はまず肌色を塗り、:large[その上に少し赤みを混ぜた肌色を重ね塗りしました。]

この辺はイラストレーターの本を読んで勉強した成果が入っているので、学んだことは別記事で紹介しようかなと思います。
